This is an example page to show how to use widgets in OGEMA. Not all widgets are shown below.

Button

This widget handles requests when clicked on it. It is straight forward to use and comes in different styles. Three of them are shown below. Clicking on one of the buttons should give you a message on the OGEMA console.

Label

Labels hold a simple Text and come in different styles.

Slider

A Slider can be used to set values without typing them in. Minimum and maximum values can be set to nearly any number. This slider posts its value when the value was changed. Different slider styles are also possible.

TextField

This widgets is mostly used in combination with other widgets. It also shows how the OGEMA widgets can be used to trigger actions on each other. Clicking the button widget below the textfield posts the input of the textfield to the OGEMA console.

Chart

A Chart can be used to display timelines or functions if given as a list of values. You can zoom in and out of the graph with mousewheel-up and mousehweel-down. Hovering over the datapoints should give you the values in a small popup window. When zoomed in you can drag the chart with the left mouse button to the left or the right.

Table

Tables are more complex than the widgets above. They have the possiblity to update their rows dynamically in a given interval. Rows inside a table are templates that are generated in Java. See TableRowTemplate.java for more information.
